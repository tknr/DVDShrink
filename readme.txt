　　　　　　【インストールとアンインストール】

　　　　　　　インストールは
　　　　　　　"dvdshrink32_jp_setup.exe"を実行してください。

　　　　　　　アンインストールは
　　　　　　　"プログラムの追加と削除"から実行してください。


　　　　　　【注意、その他】

　　　　　　　・「Win9x系」もサポートするように作者の「dvdshrink 様」に
　　　　　　　　たのんでみたところ MBCS version を作成してくださいました。
　　　　　　　　このバージョンから「Win9x系」でも使用できるようになっています。

　　　　　　　・Ver.3.1 からセットアップ形式になりました。
　　　　　　　　このインストーラーも日本語化してあります。
